API Request Examples

Create a new book (curl):

curl -X POST http://localhost:3000/api/books \
  -H "Content-Type: application/json" \
  -d '{
    "title": "The Hobbit",
    "author": "J.R.R. Tolkien",
    "genre": "Fantasy",
    "pages": 310,
    "rating": 4.7
  }'

Update a book (curl):

curl -X PUT http://localhost:3000/api/books/YOUR_BOOK_ID \
  -H "Content-Type: application/json" \
  -d '{"rating": 4.9}'

About This Project

Project Description

Book Library is a web-based platform that allows users to create and manage their personal digital book collection. Users can browse available books, add them to their library, rate and review them, and discover new books based on community recommendations. The application combines user-friendly interface design with robust backend functionality to provide a seamless reading and book discovery experience.

Technology Stack

  • Backend: Node.js with Express.js
  • Frontend: HTML5, CSS3, JavaScript
  • Database: MongoDB with native Node.js driver
  • Tools: npm, Git, VS Code
